el-grafo

el-grafo is the first development of an interactive web-based tool that allows users to intuitively explore the Bio4j domain model, getting details about how vertex and edges are shaping the network as well as precise typing information. Its purpose is to help users on their understanding of the Bio4j structure in order to manage and query it on an easier and more efficient way.
