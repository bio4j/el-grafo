el-grafo

d3-based Bio4j data model visualization

GSoC 2014 el-grafo project is the first development of an interactive web-based tool that allows users to intuitively explore the abstract domain model of the Bio4j open source bioinformatics data platform, which integrates the data available in the most representative open data sources around protein information: in UniProtKB (SwissProt + Trembl) , Gene Ontology (GO), UniRef (50, 90, 100), RefSeq, NCBI taxonomy and Expasy Enzyme.

el-grafo details the connection between all vertex and edge components which shape the logic structure of its network module by module, as well as provides specific typing information for retrieving useful data.

Its purpose is to help users gain a better understanding of the Bio4j domain model structure lowering their entry barrier to the database platform in order to utilise and query it more efficiently.

Technologically speaking, d3.js and dagre /dagre-d3 open-source JavaScript libraries compose the main core of the project.

The Bio4j database is modular, allowing users to import only the data that they are interested in. Click on each Module (colored areas) or Dependency (lines connecting them) of the scheme above to load and explore its specific graph representation.

The scheme above represents each Bio4j MODULE integrated on the database with a colored area. Its size is dependent on the number of types (vertex and edges) that shape it. Some modules require others modules to be present in the database before they can be imported. The so called Bio4j DEPENDENCIES establish connections between vertexes of different modules and are represented above by lines connecting them.

On the graph representation loaded on the right, click on each vertex to see its specific information (id and properties), and right-click on it for several filtering options via the context menu.
